# Petrol Cost Calculator

## Project Title
Smart Petrol Cost Calculator with BUDI MADANI Rebate (Malaysia)

## Description
This mobile application is developed for ICT602 Mobile Technology individual assignment. The app helps users calculate the total petrol cost based on petrol type, petrol price per liter, fuel usage, and BUDI MADANI eligibility status.

The application also calculates the BUDI MADANI rebate for eligible RON95 users and displays the final payable amount in a clear table format.

## Features
- Calculate total petrol cost
- Select petrol type: RON95, RON97, or Diesel
- Enter petrol price per liter
- Enter fuel usage in liters
- Check BUDI MADANI eligibility
- Apply RM1.99/liter rebate for eligible RON95 users
- Display calculation result in table format
- Home and About navigation
- Clickable GitHub repository link on About page

## Formula Used
Total Petrol Cost = Fuel Usage × Petrol Price per Liter

BUDI Rebate = Fuel Usage × RM1.99/liter

Final Payable Amount = Total Petrol Cost - BUDI Rebate

If the rebate is higher than the total petrol cost, the final payable amount will be limited to RM0.00.
